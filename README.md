## 개발단계

### 프론트

- 필요한 화면 구상 - 지도, 카테고리 (~ 02/22)

- 레이아웃 제작 (~ 3/01)

- 퍼블 (추후 상의)

- API 명세서 제작 후 백엔드 전달

  

### 백엔드

- 데이터 정리 및 셋팅(~3/01)

- 프론트에 어떤 데이터를 전달해야 할지



### 필요한 정보 및 기능

- 카테고리별 :  펜션,숙박 / 카페, 음식점  / 공원,놀이터  / 미용샵 / 병원 / (반려동물 고속도로 휴게소)
- 카테고리별 검색, 지역별 검색
- 마크 클릭시 제목(장소)명, 위치 제공
